# FinalProject

▶ 프로젝트 주제
   >개인을 위한 태양광 발전량 예측 모델 제공 및 웹 어플리케이션 개발    
   
   
▶ 프로젝트 내용
   > 공공 데이터를 활용한 데이터 수집 및 전처리    
   > 날씨 예측 모델 개발 (시계열 예측 모델인 LSTM과 tensorflow로 DNN 모델 사용)    
   > 태양광 발전량 예측 모델 (Light GBM 사용)    
   > AWS 배포 및 자동화 모듈    
   > 24시간 마다 공공 데이터 API를 사용해 데이터를 저장    
   
   
▶ 프로젝트 개발 환경
   >OS: Ubuntu, windows    
   >Database: Oracle DB     
   >Web: Flask
